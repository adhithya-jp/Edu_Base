EduBase - Student Management System

## Prerequisites
- Java JDK 17 or later
- Apache Maven 3.8 or later

## How to Run
1. Make sure all the source code files are in the correct directory structure inside a Maven project.
2. Open a terminal or command prompt.
3. Navigate into the root directory of the project (the one containing pom.xml).
4. Run the following command:

   mvn javafx:run

The application will build and launch. The database file `edubase.db` will be created automatically in the project's root directory.
